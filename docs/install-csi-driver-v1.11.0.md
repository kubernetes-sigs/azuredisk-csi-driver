## Install CSI driver development version on a Kubernetes cluster

### Install by kubectl
```console
curl -skSL https://raw.githubusercontent.com/kubernetes-sigs/azuredisk-csi-driver/v1.11.0/deploy/install-driver.sh | bash -s v1.11.0 --
```

 - Enable snapshot support
> Note: only available from v1.17.0
```console
curl -skSL https://raw.githubusercontent.com/kubernetes-sigs/azuredisk-csi-driver/v1.11.0/deploy/install-driver.sh | bash -s v1.11.0 snapshot --
```

### Check pods status:

```console
kubectl -n kube-system get pod -o wide --watch -l app=csi-azuredisk-controller
kubectl -n kube-system get pod -o wide --watch -l app=csi-azuredisk-node
```

